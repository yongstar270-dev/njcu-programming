## üìä Material and Methods

### 1. Data Source & Scope
* **Source:** Contents Data - Comics Part, Media Team, Oranke
* **Time Period:** April 2024 to May 2025
* **Data Process:**
    1.  Content Creation
    2.  Content Acquisition from Meta
    3.  Data Cleaning and Analysis Execution

### 2. Variables (What's in the Data)
The dataset includes both **Categorical** and **Numeric** variables, such as:

* **Categorical:** `Creator Name`, `Post Type` (Feed or Reels), `Upload Date`, `Upload Day`, `Category` (Anime, Webtoon, etc.).
* **Numeric:** `Impression`, `Reach`, `Likes`, `Shares`, `Saves`, `Total Reaction`, and `Follow`.

### 3. Optimization Techniques Used

| Strategy | Technique Used | Visualization Tool |
| :--- | :--- | :--- |
| **Day Optimization** | ANOVA Test | Tableau |
| **Time Optimization** | ANOVA Test | Tableau |
| **Creation Method Comparison** | T-test | jamovi |

## üí° Key Findings and Conclusion

### 1. Content Creation Method (Feed vs. Reels)
* **Reels** generally show a significantly higher mean for metrics like **Impression**, **Shares**, and **Comments** compared to Feed posts.
* **Feed** posts, however, show a significantly higher mean for **Saves** and **Follows**
